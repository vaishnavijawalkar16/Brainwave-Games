# Brainwave Games

Fun brain games to train your memory, focus, and reflexes. This project includes multiple mini-games designed to challenge your brain and improve cognitive skills.

## Games Included

### 1. Simon Says
- **Description:** Simon Says is a memory and pattern recognition game. It challenges your working memory by making you remember sequences of colors or sounds. It improves focus, attention, and short-term memory.
- **How to Play:**
  1. Watch carefully as the game shows you a sequence of colors (or sounds).
  2. Repeat the same sequence in the correct order.
  3. After every correct attempt, the sequence gets longer.
  4. If you make a mistake, the game ends.
- **Goal:** Remember and follow as long a sequence as possible.

### 2. Hit the Mole
- **Description:** Hit the Mole is a fast reaction game that trains your hand-eye coordination and reflexes. It improves attention, response speed, and motor skills.
- **How to Play:**
  1. The game board will have several holes.
  2. Moles will pop up randomly from these holes.
  3. Click/tap the mole quickly before it disappears.
  4. Each successful hit gives you points.
  5. As levels increase, moles appear and disappear faster.
- **Goal:** Test your reflexes and score as high as you can.

### 3. Match the Cards
- **Description:** Match the Cards is a memory game that strengthens visual memory and concentration. By remembering positions of cards, it enhances short-term memory and pattern recognition.
- **How to Play:**
  1. All cards are placed face down on the board.
  2. Flip two cards at a time to reveal them.
  3. If the two cards match, they remain face up.
  4. If they donâ€™t match, they flip back down.
  5. Continue until all pairs are matched.
- **Goal:** Use memory to match all the cards in the fewest moves.

### 4. Word Recall
- **Description:** Word Recall is a word memory game that tests your ability to memorize and recall words. It improves memory retention, attention to detail, and cognitive processing speed.
- **How to Play:**
  1. You will see a list of words on the screen for a short time.
  2. Try to memorize as many words as possible.
  3. After the words disappear, recall and type them correctly.
  4. You get points for each correct word recalled.
- **Goal:** Improve memory retention and recall speed.
